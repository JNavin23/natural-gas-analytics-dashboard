# ⛽ Natural Gas Analytics Dashboard

## 📊 Project Overview

This project presents an end-to-end Power BI dashboard built using a real-world styled messy dataset. The dashboard analyzes natural gas supply and revenue to generate meaningful business insights.

---

## 🎯 Key Objectives

* Analyze total revenue and supply performance
* Identify top-performing plants and regions
* Track revenue trends over time
* Monitor growth and efficiency metrics
* Enable interactive filtering for better decision-making

---

## 🧩 Data Preparation

* Handled missing values for revenue, region, gas type, and status
* Converted data types and cleaned inconsistencies
* Created calculated columns such as revenue per unit

---

## ⭐ Data Modeling

* Implemented a **star schema**
* Fact table: gas supply data
* Dimension tables: date, plant, region, gas type, status
* Established proper relationships for optimized performance

---

## 🧠 DAX & Calculations

The dashboard includes advanced DAX measures such as:

* Total Revenue
* Total Supply
* Average Revenue per Unit
* Year-to-Date (YTD) calculations
* Month-over-Month Growth %
* Top N and Ranking analysis
* Running total and contribution percentage

---

## 📈 Dashboard Features

* KPI cards for quick performance overview
* Revenue trend analysis over time
* Regional performance comparison
* Top performing plants visualization
* Interactive slicers (Region, Gas Type, Status, Month)
* Dynamic title based on user selection

---

## 💡 Key Insights

* Certain regions contribute significantly higher revenue
* Top plants consistently drive performance
* Revenue trend shows fluctuations with positive growth periods
* Supply and revenue efficiency varies across gas types

---

## 🛠 Tools Used

* Power BI
* Power Query
* DAX (Data Analysis Expressions)

---

## 📌 Conclusion

This project demonstrates the complete data analytics workflow, including data cleaning, modeling, and visualization. It highlights the ability to transform raw data into actionable insights using Power BI.
